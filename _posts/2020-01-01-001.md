<h2>
    A. Robot Program
</h2>
<p>
    두 경우로 나누어 생각하면 된다.
    <ul>
        <li> $$ x == y $$ 인 경우 </li>
        <li> $$ x != y $$ 인 경우 </li>
    </ul>
</p>

<div markdown="1">

```cpp
#include <iostream>

int main(void)
{
    std::cin.tie(0);
    std::ios_base::sync_with_stdio(false);

    int T;
    std::cin >> T;

    while (T--)
    {
        int x, y;
        std::cin >> x >> y;

        if( x == y ){
            std::cout << x + y << "\n";
        } else {
            std::cout << 2*std::max(x,y) - 1 << "\n";
        }
    }

    return 0;
}
```
